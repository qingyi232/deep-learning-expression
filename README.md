# 基于深度学习的表情识别系统

## 快速启动

**双击 `启动系统.bat` 文件即可运行**

## 系统简介

本系统采用深度学习CNN模型，实现7种表情的自动识别：
生气、厌恶、恐惧、开心、悲伤、惊讶、中性

## 功能特点

1. **静态图像识别** - 上传图片进行表情识别
2. **实时视频识别** - 通过摄像头实时识别表情
3. **多人脸识别** - 支持同时识别多个人脸
4. **GPU加速** - 自动使用GPU加速推理

## 使用说明

### 静态图像识别
1. 点击"静态图像识别"选项卡
2. 点击"上传图像"选择图片
3. 点击"识别表情"查看结果

### 实时视频识别
1. 点击"实时视频识别"选项卡
2. 点击"开始识别"
3. 面对摄像头做出表情
4. 点击"停止识别"结束

## 系统要求

- Python 3.7+
- NVIDIA GPU（可选）
- 摄像头（用于实时识别）

## 性能指标

- 测试集准确率：63.39%
- 推理速度：<10ms/图像（GPU）
- 模型大小：38MB

## 技术架构

- 深度学习框架：PyTorch
- 模型架构：改进VGG风格CNN
- 人脸检测：OpenCV Haar Cascade
- 图形界面：PyQt5
- 训练数据集：FER2013
